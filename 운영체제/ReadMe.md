## 프로세스와 스레드의 차이
### 프로세스(Process)
컴퓨터에서 연속적으로 실행되고 있는 컴퓨터 프로그램을 말함. 프로그램을 구동하여 프로그램 자체와 프로그램의 상태가 메모리 상에서
실행되는 작업단위를 지칭함
